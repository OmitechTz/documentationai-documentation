## Recent Releases

Stay informed about the latest enhancements in Gemba Write. We release updates frequently to improve your research writing workflow, add new features like advanced citation verification, and fix issues for smoother performance.

<Callout kind="info">
  Check your app version in the settings menu (`Settings > About`) to ensure you have the latest release. Updates deploy automatically, but restart the app to apply changes.
</Callout>

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added live quality dashboard with real-time coverage scoring and gap detection
- Introduced verified citations search across 200M+ academic sources
- Enabled structured drafting with AI alignment to research questions

## Improvements

- Enhanced export options for PDF, Word, LaTeX, and Markdown with journal-specific styles
- Optimized performance for large manuscripts (`>100 pages`)

## Bug Fixes

- Fixed citation DOI validation errors during auto-save
- Resolved export formatting issues in LaTeX mode

</Update>
