## Quick Guide

<Steps>
  <Step title="Open the docs portal">
    Visit `https://docs.gembawrite.com` in your browser.
  </Step>
  <Step title="Search for topics">
    Use the search bar at the top to find features or guides.
  </Step>
  <Step title="Browse by category">
    Click a product section in the left navigation panel.
  </Step>
  <Step title="Access API references">
    Select "API Reference" from the sidebar for integration details.
  </Step>
</Steps>
